- **Framework**: React Native with Expo
- **Navigation**: Expo Router

- **UI Library**: React Native Elements
- **Type Safety**: TypeScript
- **API**: tRPC
- **Localization**: i18next with expo-localization

## Quick Start

```bash
# Install dependencies
bun install

# Start development server
bun run dev

# Run on iOS
bun run ios

# Run on Android
bun run android
```
